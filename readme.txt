== Description ==
 
Displays and compares the environmental impact of various electric and gas vehicles based on AESO data.
 
== Installation ==
 
1. Upload plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload EV/ICE CSV files on the plugin settings page.

CSV format: vehicle_name, consumption (number), picture_url
